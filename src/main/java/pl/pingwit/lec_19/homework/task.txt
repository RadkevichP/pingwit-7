Task 1
Создайте класс Motorcycle, в котором должны быть поля brand, model, engineVolume (объем двигателя в см3),
String condition.
В классе application создайте объект типа Motocycle и сериализуйте его с помощью java-сериализации. Поле
condition сериализоваться не должно!
Далее в этом же классе десериализуйте Motocycle из файла и выведите его в консоль

Task 2
В пакете homework найдите файл atms.json
Проанализируйте этот файл и создайте класс ATM, соответствующий структуре объектов их atms.json (включите в объект те поля, которые считаете нужными)
Напишите программу, которая десереализует объекты ATM из файла в массив и выведет их в консоль.

Task 3

в Беларуси и выедет в консоль список банкоматов. Попробуйте ввести несуществующий город и сделайте так, чтобы программа
