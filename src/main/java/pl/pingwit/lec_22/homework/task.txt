Task 2 (закончите задачу с занятия)
Создайте класс Harvest, в котором будут храниться данные об урожае на ферме.
У Harvest есть поля
String field - название поля, с которого был собран урожай (урожаев за год может быть несколько собрано с одного поля)
String square - площадь посева,
String plant - название растения (рожь, пшеница, клубника, картошечка и т.д.)
BigDecimal weight - вес собранного урожая

Распарсите файл harvest.json

Создайте класс HarvestStatistic
String plant - название растения
BigDecimal totalWeight - общий вес урожая со всех полей
BigDecimal totalSquare - общая площать растения

Создайте класс HarvestStatisticService, добавьте классу метод, который будет получать аргументом коллекцию Harvest,
а возвращать коллекцию HarvestStatistic с посчитанной статистикой.

Опционально: Создайте класс, который экспортирует коллекцию HarvestStatistic в Excel
