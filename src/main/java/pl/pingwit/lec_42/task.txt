1. Напишите юнит-тесты для валидаторов.
2. Создайте в приложении get эндпоинт, который будет возвращать шутки из интренета.
Эндпоинт принимает параметр запроса count: от 1 до 5. Если параметра запроса не было (он null)
- эндпоинт должен вернуть список из 1 шутки. Если параметр больше 5 - эндпоинт дожен вернуть список из 5 шуток.
Для получения шуток используйте https://official-joke-api.appspot.com/random_joke
