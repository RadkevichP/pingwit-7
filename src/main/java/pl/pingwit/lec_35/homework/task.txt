Добавьте в проект драйвер для Postgres и библиотеку для создания Connection Pool
1. Создайте классы Product и ProductDto, соответсвующие таблице products в базе данных
2. Создайте класс ProductRepository, котором напишите методы (не статические)
    - для подсчета количества продуктов в системе
    - для поиска продукта по ID
    - для поиска продукта по имени
    - для создания продукта
    - возвращающий все продукты
    - для удаления продукта по ID

В коде методов используйте PreparedStatement
Для получения объектов типа Connection используйте объект DataSource:
ProductRepository должен иметь поле private final DataSource, которое заполняется через конструктор.
