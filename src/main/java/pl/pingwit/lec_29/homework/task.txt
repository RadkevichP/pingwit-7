Task 1
Создайте библиотеку bird-utils. В ней должен быть один класс BirdUtils:
    - метод - работа кукушки, который возьмет рандомное число от 0 до 100 и прокукует (выведет в консоль "ку-ку")
     это количество раз
    - метод - работа соловья, который прочирикает "чик-чирик"
Добавьте соберите библиотеку.

Создайте интерактивное приложение bird-listener. Приложение предложит пользователю выбрать, какую птицу слушать,
либо выйти. (все по аналогии с занятием)

Добавьте нужные плагины в pom.xml, соберите приложение в исполняемый jar-файл и протестируйте его работу из командной строки.
Отправьте jar-файл в общий чат, чтобы коллеги могли послушать птиц у себя дома.

Task 2
Добавьте в ваш проект плагины maven surefire и jacoco. Соберите ваш проект, откройте отчет о покрытии кода тестами.
Пришлите скриншоты отчета в общий чат. Изменения в проекте запушьте.
