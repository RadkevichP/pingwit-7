1. Установить Docker Desktop
2. Установить pgAdmin
3. Запустить БД Postgres, используя файл docker-compose.yml
4. Создать в pgAdmin подключение к серверу, создать в нем подключение к БД
5. В базе данных создать таблицу country и таблицу city
